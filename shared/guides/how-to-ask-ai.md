# 如何向 AI 提问

## 提问模板

```
【背景】
[简单描述你的项目/模块/上下文]

【需求】
[你想要实现的具体功能]

【已尝试】
[你已经做过什么尝试]

【问题】
[具体遇到的问题]
```

## 示例

```
【背景】
我正在开发一个 React + Antd5 的后台管理系统，
使用 pnpm 管理依赖。

【需求】
需要实现一个用户列表页面，包含搜索、表格、分页功能。

【已尝试】
已经创建了基础的页面结构，但分页功能有 bug。

【问题】
点击下一页时数据没有更新，请问可能是什么原因？
```

## 提问技巧

### 1. 提供代码上下文

```markdown
❌ "React 组件怎么写？"

✅ "我在使用 Antd5 的 Table 组件，代码如下：
```jsx
<Table dataSource={data} columns={columns} />
```
点击分页时数据不更新，如何修复？"
```

### 2. 说明技术栈版本

```markdown
✅ "使用 React 18、Antd 5.x、pnpm 8.x"
```

### 3. 提供错误信息

```markdown
❌ "代码报错"

✅ "控制台报错：`TypeError: Cannot read property 'map' of undefined`"
```

### 4. 说明目标而非步骤

```markdown
❌ "帮我写一个获取用户数据的 hook"

✅ "我需要获取用户列表数据，并在加载时显示 loading 状态，
数据回来后更新表格。如何实现这个功能？"
```

## 常用提示词

### 代码审查

```
请帮我审查以下 React 代码，检查：
1. 是否有性能问题
2. 是否有潜在的 bug
3. 是否符合最佳实践
```

### 错误排查

```
我遇到了以下错误：
[粘贴错误信息]

可能的原因是什么？如何修复？
```

### 代码解释

```
请解释以下代码的作用：
[粘贴代码]
```

### 需求转换

```
请将以下需求转换为代码实现：
[粘贴 PRD 或需求描述]
```
